MRN FOOTBALL LEGENDS uses a context-based control system.

## General Controls (Keyboard)
- **Move**: Arrow Keys or WASD
- **Sprint**: Shift

## Attacking (When you have the ball)
- **Pass (Short)**: Z
- **Shoot**: X
- **Through Ball**: C
- **Skill Move**: D

## Defending (When opponent has the ball)
- **Press**: Z
- **Tackle**: X
- **Slide Tackle**: C
- **Switch Player**: D

## Set Pieces
- **Aim**: Arrow Keys
- **Power**: Hold Shoot (X)
- **Curve**: Hold C + Arrow Keys

> Note: Connect a gamepad for the best experience (Support coming soon).
